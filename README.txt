CMSC434: Android Doodler App
By: Al-Latif Alston
UID: 112326920

How to run: Simply install and click the "Doodler" app.
Custom features:
- Saturation slider
- Eraser button
- Fill button

The saturation slider simply controls the level of saturation in the current color.
The eraser button changes the brush to an eraser, works with brush size.
The fill button fills the entire canvas with the current color.
